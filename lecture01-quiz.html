Q1この科目の3つの狙いに含まれないものはどれ？
AAIを効果的に使って資料を作れること
BAIを使ってデータ分析ができること

この科目はエンジニアを育てる科目ではありません。3つの狙いは「①AI×資料作成 ②AI×データ分析 ③コードが読める価値の実感」です。
Q2AIユーザーの4段階で「AIの答えを疑える」のはレベルいくつ？
Aレベル1
Bレベル2
Cレベル3
Dレベル4
正解: B
レベル1=AIに聞ける、レベル2=AIの答えを疑える、レベル3=AIを正確に操れる、レベル4=AIで仕組みを作れる。この科目はレベル2〜3を目指します。
Q3バイブコーディング（Vibe Coding）の説明として正しいのはどれ？
AAIが自動的にバグを見つけて修正する技術
Bプログラミング言語を使わずにGUIだけで開発する手法
C人間が日本語で指示し、AIにコードを書かせるアプローチ
DAIが既存のプログラムを勝手に書き換え続ける仕組み
正解: C
バイブコーディングは2025年にAndrej Karpathyが提唱した概念で、自然言語でAIに指示を出し、AIにプログラムを書かせるアプローチです。
Q4この科目で扱うAIの中核技術はどれ？
A大規模言語モデル（LLM）
BルールベースAI
C画像認識AI
D音声認識AI
正解: A
AI → 機械学習 → 深層学習 → 生成AI → LLM。この科目はLLMベースの生成AIツール（ChatGPT、Gemini、Claude等）を使います。

Q1Google Colabでコードを実行すると、処理はどこで行われる？
A自分のパソコン
B大学のサーバー
Cブラウザ自身の中（JavaScriptとして実行）
DGoogleのサーバー（クラウド）
正解: D
Colabはブラウザが入出力の窓口で、実際の計算はGoogleのサーバーが担当します。そのためインターネット接続が必要です。
Q2Colabの「コードセル」と「テキストセル」の説明として正しいのはどれ？
AコードセルはPythonを実行する場所で、テキストセルはメモを書く場所
Bコードセルはメモを書く場所で、テキストセルはプログラムを実行する場所
CコードセルはPython専用、テキストセルはExcel関数を書く場所
Dどちらも同じ機能で、名前が違うだけ
正解: A
コードセルはPythonコードを書いて実行する場所（左に▶ボタン）、テキストセルはメモや説明を書く場所（Markdown形式）です。
Q3print("Hello World") を実行すると何が表示される？
A"Hello World"（ダブルクォーテーション付き）
Bprint(Hello World)
CHello World
Dエラーになり、何も表示されない
正解: C
print() は括弧内の文字列を画面に表示する関数です。ダブルクォーテーションは文字列を囲む記号であり、出力には含まれません。
Q4セルを実行して次のセルに移動するショートカットはどれ？
ACtrl + C
BShift + Enter
CCtrl + Enter
DAlt + Enter
正解: B
Shift + Enter でセルを実行して次のセルに移動します。Ctrl + Enter はそのセルに留まり、Alt + Enter は新しいセルを挿入します。

Q1この科目で使う4つのAIツールの組み合わせとして正しいのはどれ？
AChatGPT、Gemini、Siri、Perplexity
BChatGPT、Alexa、Claude、Perplexity
CChatGPT、Gemini、Claude、Perplexity
DChatGPT、Copilot、Claude、Bard
正解: C
この科目ではChatGPT（OpenAI）、Gemini（Google）、Claude（Anthropic）、Perplexity（Perplexity AI）の4つを使います。
Q2同じ質問を複数のAIに投げたとき、回答が異なる主な理由はどれ？
A各AIが異なるデータで学習し、異なるアルゴリズムで回答を生成しているから
BAIがわざと違う答えを返すようプログラムされている
C質問するたびにランダムな答えを返す仕組みだから
Dユーザーの位置情報によって回答が変わるから
正解: A
LLMは学習データとアルゴリズムが異なれば出力も異なります。友人AとBで「おすすめのラーメン屋」が違うのと同じです。
Q3回答に出典（ソース）が付くのが特徴のAIツールはどれ？
AChatGPT
BClaude
CGemini
DPerplexity
正解: D
PerplexityはAI検索エンジンであり、回答に出典リンクが付きます。レポートの参考文献探しに便利です。
Q4AIの回答について、この科目で身につけたい力はどれ？
AAIの回答をそのまま信じて使う力
BAIの回答が正しいか検証できる力
CAIに質問する回数を減らす力
DAIよりも速く自分で答えを出す力
正解: B
AIの回答は「正解」とは限りません。「AIの答えを疑える」（レベル2）がこの科目の目標です。
