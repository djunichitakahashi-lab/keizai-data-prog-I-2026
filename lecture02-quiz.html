Q1CSVファイルの説明として正しいのはどれ？
Aバイナリ形式（人が読めない形）で保存されたファイル
BExcelでしか開けない専用ファイル
Cカンマ（,）でデータを区切ったテキストファイル
D色や罫線でデータを装飾できるファイル形式
正解: C
CSVはComma-Separated Values（カンマ区切り値）の略で、メモ帳でも開けるシンプルなテキストファイルです。
Q2CSVとExcelを比べたとき、AIにデータを渡す場合に有利なのはどちら？
AExcel — 数式やグラフが含まれるので情報量が多い
BPDF — 表形式で見やすいので最適
CCSV — テキスト形式なのでAIが読みやすい
Dどちらも同じ — AIにとって違いはない
正解: C
CSVはテキスト形式なのでAIが直接読み込みやすく、最も確実にデータを渡せる形式です。
Q3CSVファイルの「ヘッダー」とは何のこと？
A1行目にある列の名前（sex, weight, height など）
Bファイル末尾にある電子署名
Cデータの最終行にある合計値
Dファイルの作成日時を記録した部分
正解: A
ヘッダーはCSVの1行目にある列名のことです。Davisデータでは sex, weight, height, repwt, repht がヘッダーです。
Q4Davisデータセットの説明として正しいのはどれ？
A日本のGDP推移を記録した経済データ
B運動習慣のある男女200人の身長・体重・自己申告値のデータ
C世界の気温と降水量のデータ
D大学生のスマートフォン利用時間データ
正解: B
Davisデータは運動習慣のある男女200人の実測体重・実測身長・自己申告体重・自己申告身長を記録したデータセットです。
Q5「欠損値」とは何のこと？
Aデータが記録されていない（抜けている）箇所
B値が0になっているデータ
C大文字英字で入力された値
D小数点を含むデータ
正解: A
欠損値（missing value）はデータが記録されていない箇所のことです。Davisデータではrepwtとrephtに欠損があり、CSVではNAや空欄で表されます。
Q6Davisデータセットの行数と列数の組み合わせとして正しいのはどれ？
A200行 × 5列（sex, weight, height, repwt, repht）
B1000行 × 3列
C500行 × 10列
D100行 × 4列
正解: A
Davisデータは200人分の身体データで、列はsex（性別）、weight（実測体重）、height（実測身長）、repwt（自己申告体重）、repht（自己申告身長）の5列です。なおCSVには行番号列が付いているため、AIによっては「6列」と報告することがあります。

Q1CSVファイルをアップロードしてAIに分析させる手順として正しいのはどれ？
AAIにメールでCSVを送信する
BAIにファイル名だけ伝えれば自動で見つけてくれる
CCSVをダウンロード → AIにアップロード → 指示文を送信
DCSVの中身をすべて手入力でチャットに貼り付ける

ファイルアップロード方式では、CSVをダウンロードし、AIの📎アイコンからアップロードし、指示文を送ります。（STEP 4ではURLを直接渡す別の方法も学びます。）
Q2AIへの指示文のことを何と呼ぶ？
Aソースコード
Bアルゴリズム
Cコマンド
Dプロンプト
正解: D
AIに何かを頼むときの指示文を「プロンプト」と呼びます。具体的に書くほど具体的な答えが返ってきます。
Q3Davisデータの有名な「外れ値」とはどんなデータ？
A身長と体重が入れ替わっているように見えるデータ
B性別が記録されていないデータ
C同じ値が連続して入力されているデータ
Dすべての列が空欄のデータ
正解: A
ある1人のデータで身長57cm・体重166kgと記録されており、身長と体重が入れ替わっているように見える有名な外れ値です。
Q4AIに「このデータを分析して」と渡したとき、AIが返してくれる内容に含まれないのはどれ？
A基本統計量（平均値・中央値など）
B欠損値の有無と件数
Cデータを収集した研究者の連絡先
D散布図やヒストグラムなどの可視化
正解: C
AIはデータの統計量・欠損値・外れ値などを分析しますが、研究者の連絡先のようなメタ情報は通常返しません。
Q5AIへのプロンプトとしてより具体的で良い例はどれ？
A「いい感じにグラフ描いて」
B「教えて」
C「分析して」
D「男女別に体重と身長の平均値を計算して、表で示してください」
正解: D
プロンプトは具体的であるほど具体的な回答が得られます。「何を」「どのように」「どんな形式で」を含めると、AIは欲しい結果を返してくれます。

Q1同じデータ・同じプロンプトでも複数のAIで結果が異なる主な理由はどれ？
AAIの1つが間違っているから
B各AIが異なるデータで学習し、異なるアルゴリズムで回答を生成しているから
C質問する時間帯が違うから
DCSVファイルがAIごとに異なる内容に変換されるから
正解: B
各AIは異なる学習データ・アルゴリズムを持っているため、同じ入力でも着目するポイントや出力するグラフが異なります。
Q2AIの分析結果について、この科目で大事にしたい姿勢はどれ？
A複数のAIで比較し、結果を鵜呑みにせず検証する
BAIの回答はすべて正しいのでそのまま使えばよい
C一番信頼できるAIを1つ決めて、そのAIだけ使う
DAIには触れない方が安全
正解: A
AIの回答は「参考意見」であり「正解」ではありません。複数AIで比較することで偏りに気づけます。
Q3「散布図」で分かることはどれ？
A2つの変数の関係（例：身長と体重の関係）
Bデータの時系列変化
C1つの変数のばらつき
Dデータ全体の平均と分散
正解: A
散布図は横軸と縦軸に2つの変数をとり、データの関係を点で表すグラフです。例えば身長×体重の関係が一目で分かります。
Q4AIが散布図に引いた線のことを何と呼ぶ？
A回帰線
B中央線
C平均線
D補助線
正解: A
散布図のデータの傾向を直線で表したものが「回帰線」です。第11回で回帰分析として詳しく学びます。
Q5ChatGPT・Gemini・Claudeの提供元の組み合わせとして正しいのはどれ？
AChatGPT=Microsoft／Gemini=Apple／Claude=Meta
BChatGPT=OpenAI ／ Gemini=Google ／ Claude=Anthropic
CChatGPT=Google ／ Gemini=Anthropic ／ Claude=OpenAI
D3つともGoogleが提供している
正解: B
ChatGPTはOpenAI、GeminiはGoogle、ClaudeはAnthropicがそれぞれ独自に開発しています。学習データもアルゴリズムも違うため、同じプロンプトでも結果が変わります。

Q1CSVのURLをプロンプトに貼ってAIに渡す方法のメリットはどれ？
A非公開データも含めてすべてのデータを渡せる
Bダウンロード＆アップロードの手間が省ける
CAIの分析精度がアップロードより高くなる
Dファイルサイズが大きくても確実に渡せる
正解: B
URLをプロンプトに貼るだけでOKなので、ファイルをダウンロードしてアップロードする手間が省けます。ただし公開データのみ使える方法です。
Q2ChatGPTのデータ分析機能で、裏で動いていることがあるのはどれ？
A電卓アプリ
BPythonコード
C検索エンジン
DExcelのマクロ
正解: B
ChatGPT・Gemini・Claude・PerplexityなどはどれもPythonコードを実行してグラフを描けます。コードを見たいときは「使ったコードを見せてください」とテキストで頼むのが一番確実。〈〉ボタンが出ていればそれを押すだけでも見られます。
Q3Pythonコードの import pandas as pd は何をしている？
A「pandas」という名前のデータを画面に表示している
Bデータ分析用の道具箱（ライブラリ）を読み込んでいる
CPDFをpandasという形式に変換している
DCSVファイルをインポート（取り込み）している
正解: B
import は道具箱を読み込む命令です。pandas はデータ分析に使うPythonのライブラリで、pd という短い名前を付けて使います。
Q4Pythonコードに出てくる df は何を指している？
AAIモデルの設定値
BCSVファイルの名前
Cダウンロードした画像ファイル
Dデータの入れ物（DataFrame）
正解: D
df はDataFrame（データフレーム）の略で、表形式のデータを格納する入れ物です。Pythonでデータ分析するときによく使われる変数名です。
Q5Pythonコードの print(df.describe()) は何をする命令？
AdfをCSVファイルに保存する
Bdfの中身をExcelで開く
Cデータの基本統計量（件数・平均・最小・最大など）を画面に表示する
D散布図やヒストグラムを描く
正解: C
df.describe() は基本統計量（件数・平均・標準偏差・最小・四分位数・最大）を計算し、print() がそれを画面に表示します。AIが返してくる「基本統計量の表」は、裏側でこのコードが動いていることが多いです。

Q1Google Colabの説明として正しいのはどれ？
A有料のプログラミングソフト
BChatGPTのようなAIチャットツール
C写真や動画を編集する Web アプリ
Dブラウザ上でPythonを動かせる無料ツール
正解: D
Google ColabはGoogleが提供する無料のPython実行環境です。ブラウザだけで動き、インストールは不要です。
Q2Colabの「コードセル」と「テキストセル」の違いとして正しいのはどれ？
Aテキストセルでも Python が実行できる
B見た目が違うだけで機能は同じ
Cコードセルはメモ用、テキストセルはPython実行用
DコードセルはPythonを実行する場所、テキストセルはメモを書く場所
正解: D
コードセルは左に▶ボタンがあり、Pythonコードを書いて実行できます。テキストセルはMarkdown形式でメモを書く場所です。
Q3Colabでコードを実行したとき赤い文字（エラー）が出た場合、どうすべき？
Aエラーは無視して次のセルを実行する
Bエラーの最後の1行を読んで原因を確認し、修正して再実行する
Cノートブックを新規作成して最初からやり直す
Dパソコンが壊れる可能性があるので即座にブラウザを閉じる
正解: B
エラーは「次の手がかり」です。最後の1行にエラーの原因が書いてあるので、そこを読んで修正→再実行するのがプログラミングの日常です。
Q4AIが出力したコードをColabで動かすとき、正しい手順はどれ？
Aコードをテキストセルに貼り付けて保存する
Bコードを印刷してから手で打ち込む
CAIにコードを表示してもらう（テキスト依頼か〈〉ボタン）→ Colabのコードセルに貼り付け → ▶で実行
Dコードの内容を暗記してから自分で書き直す
正解: C
AIに「使ったコードを見せて」とテキストで頼むのが一番確実（〈〉ボタンが出ていればそれを押してもよい）。コピーしたら Colab のコードセル（左に▶ボタンがある方）に貼り付け、▶ または Shift+Enter で実行します。同じ結果が自分の手で得られます。
Q5Colabのコードセルを実行するキーボードショートカットはどれ？
ACtrl+S（コマンド+S）
BTab キー
CAlt+F4
DShift+Enter（実行して次のセルへ進む）
正解: D
Shift+Enter で現在のセルを実行し、次のセルにフォーカスが移ります。▶ ボタンを押すのと同じ操作です。Ctrl+Enter は実行後にそのセルに留まります。
